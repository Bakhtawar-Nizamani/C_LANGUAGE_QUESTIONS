# Conditional Statements in C (IF / ELSE)

This repository contains a collection of **10 basic C programming problems** based on conditional statements (`if`, `else if`, `else`).

I created this to practice and improve my **logic building and problem-solving skills** as I continue learning programming.


# Problems Covered

Here are the problems included in this repo:

1. Check whether a number is **even or odd**
2. Check whether a number is **positive, negative, or zero**
3. Find the **largest of three numbers**
4. Check whether a year is a **leap year**
5. Check whether a number is **divisible by 5 and 11**
6. Check whether a character is a **vowel or consonant**
7. Calculate **student grade** based on marks
8. Check whether a number is **prime**
9. Check whether a number is a **palindrome**
10. Calculate **electricity bill using slabs**

# Tech Stack

* Language: **C**
* Compiler: GCC C compiler
