# Loops in C (for / while / do-while)

This section contains a collection of **10 C programming problems** based on loops.
These problems help in understanding how to use iteration effectively and build stronger logic.

# Problems Covered

11. Print numbers from **1 to N**
12. Print all **even numbers between 1 and N**
13. Find the **sum of first N natural numbers**
14. Print **multiplication table** of a number
15. Count the **digits in a number**
16. **Reverse a number**
17. Find the **factorial** of a number
18. Print the **Fibonacci series**
19. Find the **GCD of two numbers**
20. Check whether a number is an **Armstrong number**


# Tech Stack

* Language: **C**
* Concepts: **for loop, while loop, do-while loop**
