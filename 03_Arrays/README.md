# Arrays in C (1D, 2D & Nested)

This section contains a collection of **10 C programming problems** based on arrays.
These problems help in understanding how to store, access, and manipulate data using **1D and 2D arrays (matrices)**.

# Problems Covered

21. Find the **largest and smallest element** in an array
22. **Reverse an array**
23. Find the **sum and average** of array elements
24. Count **even and odd numbers** in an array
25. **Search an element** using linear search
26. **Sort an array** in ascending order
27. Add two **1D arrays**
28. Add two **2D matrices**
29. Find the **transpose of a matrix**
30. Find the **sum of each row and column** of a matrix


# Tech Stack

* Language: **C**
* Concepts: **Arrays (1D & 2D), nested loops**
