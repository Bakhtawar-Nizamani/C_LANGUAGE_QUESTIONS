# Functions in C 

This section contains a collection of **10 problems based on functions** in C.
These problems focus on breaking code into smaller parts and improving **modularity, reusability, and clean coding practices**.


# Problems Covered

41. Function to check **prime number**
42. Function to calculate **factorial**
43. Function to find **maximum of two numbers**
44. Function to **reverse a number**
45. Function to check **palindrome**
46. Function to find **sum of array elements**
47. Function to **swap two numbers (call by value)**
48. Function to **swap two numbers (call by reference)**
49. Recursive function for **factorial**
50. Recursive function for **Fibonacci series**


# Tech Stack

* Language: **C**
* Concepts: **functions, parameters, return values, recursion**
